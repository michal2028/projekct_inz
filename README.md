# Footbal Scores Application

It's a college credit application. It is used to present the use of an external API on the example of match statistics.

## Technologies
* HTML5
* CSS3
* SASS
* JavaScript
* AJAX
* animate.css 4.1.1
* live-server 1.2.2
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../project_inz
$ npm  run build
$ npm  run compile
```
